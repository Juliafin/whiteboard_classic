### Teacher Manager


## Purpose 

# A manager of curriculum and student info for teachers. And a way for students to keep track of their projects and current progress, and lesson times. 

## Technologies used: 

# Frontend

- HTML5
- CSS3
- JavaScript
- JQuery


# Backend

- Node.js
- Express.js
- Faker.js
- MongoDB
- Mongoose.js
- JSON web tokens
- Bcrypt.js
- JavaSript (ES6)
- Mocha.js
- Chai.js
